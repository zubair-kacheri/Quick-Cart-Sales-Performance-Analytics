# Quick-Cart-Sales-Performance-Analytics – Python Data Analysis Project

## Project Overview

This project presents an end-to-end data analysis of **QuickCart retail sales data** using Python.
The objective is to evaluate sales performance, product behavior, outlet efficiency, and customer satisfaction through structured exploratory data analysis and visualizations.

The analysis is implemented in a Jupyter Notebook to demonstrate practical data analytics skills suitable for real-world business scenarios.

---

## Business Objectives

* Understand overall sales performance and revenue drivers
* Analyze product categories and fat content impact on sales
* Evaluate outlet performance based on size, location, and age
* Identify customer rating trends
* Provide data-driven insights for business optimization

---

## Dataset Summary

The dataset includes the following information:

| Category            | Fields                                                      |
| ------------------- | ----------------------------------------------------------- |
| Product Details     | Item Type, Fat Content, Visibility, Price                   |
| Outlet Details      | Outlet Type, Outlet Size, Location Tier, Establishment Year |
| Performance Metrics | Sales, Customer Rating                                      |

---

## Tools & Technologies

* Python
* Jupyter Notebook
* pandas – data manipulation and transformation
* numpy – numerical operations
* matplotlib & seaborn – visual analytics

---

## Analytical Workflow

1. Data Loading & Inspection
2. Data Cleaning and Handling Missing Values
3. Feature Engineering
4. Exploratory Data Analysis (EDA)
5. GroupBy & Aggregation Techniques
6. Visualization of Trends and Patterns
7. Business Insight Generation

---

## Key Insights

* High-fat content products contribute a larger share of total revenue.
* Medium-sized outlets exhibit stronger average sales performance.
* Tier-1 locations consistently outperform Tier-2 and Tier-3 regions.
* Older outlets demonstrate stable sales trends over time.
* Certain product categories drive a disproportionate share of revenue.
